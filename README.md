# Onyx Launcher

Modern Minecraft Launcher for Modpack Creators & Small Communities.

## Overview

Onyx Launcher は、小規模コミュニティや身内サーバー向けに設計された Minecraft ランチャーです。

Minecraft 本体の管理だけでなく、

* Forge
* Fabric
* Modrinth

との連携を行い、MOD構成の配布や共有を簡単に行うことを目的としています。

複雑なセットアップを必要とせず、誰でも同じ環境を再現できるよう設計されています。

---

## Features

### Instance Management

* 複数の起動構成を管理
* Forge / Fabric 対応
* 構成ごとの設定保存
* 構成のインポート / エクスポート

### Modrinth Integration

* MOD検索
* ワンクリック導入
* 依存MODの自動解決
* MOD情報の表示

### Easy Distribution

* 構成共有機能
* ZIP形式での配布
* 初心者でも簡単に導入可能

### Microsoft Login

* Microsoftアカウント対応
* オフラインモード対応

### Modern UI

* ダークモード
* ライトモード
* カスタムアクセントカラー
* モダンなランチャーデザイン
